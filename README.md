# ระบบประมวลผลข้อมูลงานแนะแนว วษท.เชียงใหม่

เว็บแอปแสดงผลและวิเคราะห์ข้อมูลงานแนะแนว วิทยาลัยเกษตรและเทคโนโลยีเชียงใหม่
ทำงานแบบ **static** ไม่ต้องติดตั้งอะไร ไม่ต้องต่ออินเทอร์เน็ต (ฟอนต์สวยขึ้นเมื่อออนไลน์)

## วิธีเปิดใช้งาน
- **ง่ายที่สุด:** ดับเบิลคลิกไฟล์ `index.html` เปิดในเบราว์เซอร์ได้ทันที
- **หรือเปิดผ่านเซิร์ฟเวอร์:** `node serve.js` แล้วเปิด http://localhost:8090

## เนื้อหา 7 ส่วน
1. **ภาพรวม** — ตัวชี้วัดหลัก (KPI) และบทสรุป
2. **จำนวนนักเรียน** — แนวโน้ม นร.นศ. 10 ปี (2560–2569)
3. **การรับสมัคร 2569** — 3 รอบ แยก ปวช./ปวส. และช่องทางสมัคร
4. **กลุ่มเป้าหมาย** — ผลแบบสำรวจการตัดสินใจเข้าเรียน (ช่องทาง/ผู้มีอิทธิพล/เหตุผล)
5. **ภาวะผู้จบ** — ศึกษาต่อ/มีงานทำ/รองาน รายปี–รายสาขา
6. **ผู้ปฏิบัติงาน** — ทีมงานแนะแนว
7. **วิเคราะห์เพื่ออนาคต** — จุดแข็ง/ความเสี่ยง/ข้อเสนอเชิงกลยุทธ์

## การแก้ไข/อัปเดตข้อมูล
แก้ไฟล์เดียว: **`assets/js/data.js`** แล้วรีเฟรชหน้าเว็บ กราฟและตารางจะอัปเดตเอง
- เพิ่มปีการศึกษา → เพิ่มบรรทัดใน `enrollment`
- อัปเดตยอดรับสมัคร → แก้ `admission2569`
- แก้รายชื่อทีมงานจริง → แก้ `staff` (ปัจจุบันเป็น **ตัวอย่าง**)

## ที่มาข้อมูล
สร้างจากไฟล์ต้นฉบับในโฟลเดอร์แม่: ยอด นร.นศ. ปี 60–69, ประกาศรายชื่อ 3 รอบ ปี 69,
แบบสำรวจการตัดสินใจเข้าเรียน, และรายงานภาวะการมีงานทำผู้จบปี 66/67/68
